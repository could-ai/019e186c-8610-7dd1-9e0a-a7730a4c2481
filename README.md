# Presentazione: Marketing e Social Media nell'Era dell'IA

Questo progetto è una presentazione interattiva in stile PowerPoint/Keynote sviluppata con Flutter, pensata per un'esposizione di 50 minuti sul marketing nei social media (TikTok, Instagram, YouTube, LinkedIn) e sull'uso dell'Intelligenza Artificiale (come la creazione di modelli virtuali).

## Funzionalità
- **Modalità Presentazione:** Interfaccia a tutto schermo ottimizzata per schermi grandi e proiettori.
- **Navigazione da Tastiera:** Usa le frecce direzionali (Destra/Sinistra) o la barra spaziatrice per muoverti tra le slide.
- **Contenuti Integrati:** Slide strutturate per coprire gli argomenti chiave, aneddoti e placeholder per video esplicativi.
- **Design Minimalista:** Nessun elemento tipico delle app (niente AppBar o bottoni invadenti), per garantire la massima concentrazione sui contenuti.

## Come eseguire la presentazione
Assicurati di avere [Flutter](https://docs.flutter.dev/get-started/install) installato sul tuo sistema.

1. Apri un terminale nella cartella del progetto.
2. Esegui `flutter pub get` per scaricare eventuali dipendenze.
3. Esegui `flutter run -d macos` (o `windows`, `linux`, `chrome` a seconda del tuo sistema) per avviare la presentazione a schermo intero.

## Struttura del Progetto
- `lib/main.dart`: Contiene l'engine della presentazione, il sistema di navigazione via tastiera e tutte le singole slide.
